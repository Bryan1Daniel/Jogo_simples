# 🕹️ Amostra Jogo Dan (Meu "Hello World" em Game Dev)

> **Aviso de Sinceridade:** Este projeto é simples, visualmente "bobo" e foi feito puramente para uma apresentação escolar em 2025. Decidi mantê-lo aqui como um registro histórico do meu primeiro contato com lógica de jogos.

### 📝 O que é isso?
Este foi um experimento feito com a biblioteca **p5.js**. A ideia era apenas criar algo funcional que pudesse ser mostrado no colégio. Não espere gráficos de última geração ou uma jogabilidade complexa — o foco aqui era simplesmente fazer o código rodar e a lógica funcionar.

### 🛠️ Tecnologias "Raiz"
* **JavaScript + p5.js**: Para a lógica e renderização dos desenhos.
* **HTML/CSS**: Apenas para segurar o jogo na tela.

### 🧠 O que eu aprendi (apesar da simplicidade):
Mesmo sendo um projeto básico, ele me ensinou coisas que uso até hoje:
- Como funciona um **Game Loop** (o computador desenhando várias vezes por segundo).
- Detecção de colisão (fazer o código entender quando dois objetos se encostam).
- Gerenciamento de estado (início, jogo e "game over").
